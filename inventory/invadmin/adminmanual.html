Welcome to admin system manual

The inventory system is protect with 3 levels of account Rola Based Access

  • Admin (Have full access the whole system to modify all posisble data)
  • Sales Lead (Have access to the Stock , Product and Sales desk able to adjust and update)
  • Cashier (Have Access to Sales desk both cash / credit sale)
  • Cashual (Have no access to system can not login)
Only for admin =>> Customer , Business, Analysis Dashboard and Reports All new creations in the system are recorded and once you login system tracks you till you log outM

On this page only user whom are allowed to use the system and have account credentials are able to fill the form details

  • Username (in this case user email registered in the system)
  • Password (in this case ID number of newly created account should be changed)
With correct password and username will be able to access the system accoding to privilege given by Admin.

Only Admins are able to create and add new staff to the system and this following are to fill must (ID Number , Name , Email ,Phone, Role ). Ensure you fill all the details correctly Note : incase of incorrect details this can be corrected on Adjust Staff Tab By creating new staff will be able to login immediately to the system and also once you create new Staff you will be loged in in His or Her. With this you can check the privilege since all the access will be shown

This is the firsr item to add to the system: Stock is entered and deduction will be done creation of product.

  • CODE /ID (This is unique to each stock and should never be reused)
  • Measure (Metrics used to measure the stock eg Kgs, L )
  • Unit count (Metrics of the stock in eg 1 dozen, 100 kgs)
  • Unit to sale (items to be on the shelf eg 12 for 1 dozen or 100 for 100 kgs if you will sale only minimum of 1 kg)
Note : Clear date if both for expire and for non expiry Items it is used to indicate how long can it be in stock

Only product can be created form stock you must search stock first and create product out of it use CODE/ ID

  • CODE /ID (This is unique to each stock and should never be reused or be same as for stock to avoid confusion)
  • Selling/ Buying (Must be filled for sales desk oparations)
  • Unit to sale (Metrics from stock to the shelf)
  • Unit in stock (Never change this number it is automaticaly calculated the reminder)
Note : Expiry and clear date are automaticaly picked from the stock indication
